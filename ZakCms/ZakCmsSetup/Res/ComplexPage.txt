## Description
Create a new context configuration for a given context and target type.

There can only be one configuration for a given context and target type combination. 
The system will return an error if this method is called and a configuration 
already exists for the given context and target type.

If a configuration already exists for the context and target type, 
it's required to delete the current configuration prior to creating a new one. 
Alternatively, the Update method may be used. 

## Request URL
http://acme.com/api/contexts/configurations/*targetType*

## HTTP Verb
POST

## Content Type
application/json

## Parameters

### Uri Segment
  - **targetType**:
	- **0** (or **ios**) for iOS
	- **1** (or **android**) for Android
	- **2** (or **wp7**) for Windows Phone 7
	- **3** (or **rim**) for RIM

### Uri Query String
*None*

### Body
The configuration appropriate for the given target type. 
See the Get method for the detailed structure.


### Response Data
If the call is not successful, the "Code" property of the body will have a value > 0 
and an error message in the "Message" property; otherwise, if the call is successful, 
the response contains a "Result" property containing the newly 
created context configuration.
